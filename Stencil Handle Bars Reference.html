Handlebars Helpers Reference

{{itemAt}}

Block helper that returns the item at the specified index.

Parameters

  • array {Array}
  • id {Number}
  • array {Array}
  • returns {any} value

Example

Given the array ['a', 'b', 'c']:

{{itemAt array 1}}

//=> 'b'
